# Utility Scripts

This folder contains small helper scripts used during the project.

## Files

| File | Purpose |
| --- | --- |
| `data_test.py` | Utility check related to dataset handling |
| `filesize.py` | Utility for checking artifact size or footprint |

## Why This Folder Exists

These scripts are not the main experiments. They support reproducibility and reporting by handling small, reusable checks outside the notebooks.
